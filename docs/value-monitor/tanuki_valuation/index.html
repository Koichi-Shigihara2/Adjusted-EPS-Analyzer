METHODOLOGY

V₀ 本質的価値

過去5年平均FCFを起点に、2段階DCFで算出。 高成長期（3年）はFCF CAGRを適用し、ターミナル成長率3%で永続価値を計算。
WACC = 8.5%

α 成長期待プレミアム

直近連続黒字期間のROE平均 × 内部留保率から個別成長率を算出。
α = (g / WACC) × 0.7
αは1.0でキャップ。

RPO補正

SaaS企業向け。Remaining Performance Obligations（残存履行義務）を 15%割引率で現在価値化し、V₀に加算。

1〜3年後予測

高成長期間中は企業別CAGRを適用、以降はターミナル成長率。 本質的価値の将来推移を自動計算。
